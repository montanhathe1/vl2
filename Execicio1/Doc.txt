Objetivo: 
Montar uma calculadora que some dois números

Requisitos obrigatórios:
1. Aplicação roda no console
2. O resultado deverá ser exibido logo após o comando de execução do cálculo
3. Todas operações, bem como seus resultados deverão ser guardados em um a lista (esta lista não precisa ser exibida)

Requisitos não obrigatórios
1. Ao entrar na aplicação ela solicita os dois números para soma
2. Apos mostrar os resultado, a aplicação retorno ao estado inicial
3. A aplicação é encerrada digitando x como primeiro número
4. Antes de sair a aplicação exibir a lista de operação

Arquitetura:
Linguagem: Java
Repositório: GitHub (Cria uma pasta para esse exercício)
Tempo de execução máximo: 2 dias
